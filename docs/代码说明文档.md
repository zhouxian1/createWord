# 代码说明文档

## 1. 系统概述

本项目是面向 GJB 438C 军用软件文档编制的文档生成与知识管理系统。系统通过文档解析、知识库索引、知识图谱、规则验证和本地大模型生成，支撑从资料导入到文档交付的完整流程。

代码分为后端 `backend` 和前端 `frontend` 两部分。后端负责业务规则、数据持久化、文档处理、模型调用和 REST API；前端负责页面展示、交互流程和接口调用。

## 2. 技术栈

### 后端

- Flask 3：Web 框架。
- Flask-SQLAlchemy：关系数据模型和数据库访问。
- SQLite / PostgreSQL：业务数据存储，开发环境默认 SQLite。
- ChromaDB：向量检索，可使用本地持久化或独立服务。
- Neo4j：知识图谱持久化，可选。
- MinIO：对象存储，可选，不可用时回退本地文件。
- sentence-transformers：中文向量模型。
- jieba：中文分词与倒排索引。
- python-docx、pdfplumber、PyPDF2、openpyxl、python-pptx：文档解析和生成。
- requests：Ollama 原生接口调用。

### 前端

- Vue 3 + Vite 5
- Vue Router
- Pinia
- Element Plus
- Axios
- D3.js
- ECharts / vue-echarts

## 5. 核心模块说明

### 5.1 LLM 服务

文件：`backend/app/services/llm_service.py`

当前默认模型服务为 Ollama：

```text
provider: ollama
api_base: http://192.168.31.245:11434
model: qwen2.5:7b
endpoint: /api/generate
```

`LLMService.generate()` 会根据 provider 分流：

- `ollama`：调用 Ollama 原生 `POST /api/generate`，请求体包含 `model`、`prompt`、`stream=false` 和 `options`。
- 其他 provider：保留 OpenAI 兼容接口调用能力。

`create_llm_service()` 默认返回 `OllamaService`，用于避免旧环境变量误把服务切回 DashScope。

### 5.2 问答服务

文件：`backend/app/services/qa_service.py`

职责：

- 通用知识库问答：检索知识库上下文后调用 LLM。
- 438C 专项问答：基于内置 438C 标准摘要构造 prompt 后调用 LLM。
- 返回 source trace，支持前端展示溯源。

438C 专项问答已限制上下文文档数量和章节数量，避免 `qwen2.5:7b` 在超长 prompt 下响应过慢。

### 5.3 文档生成服务

文件：`backend/app/services/document_generator.py`

职责：

- 创建 438C 文档骨架。
- 按整篇或章节生成内容。
- 基于代码文件生成文档片段。
- 记录生成任务状态。

默认通过 `create_llm_service()` 调用本地 Ollama 模型。

### 5.4 知识库和检索

主要文件：

- `document_parser.py`
- `military_doc_chunker.py`
- `semantic_indexer.py`
- `inverted_index.py`
- `rerank_service.py`

流程：

1. 文件解析为文本和元数据。
2. 军标文档按章节、段落、要素拆分。
3. 写入语义索引和倒排索引。
4. 检索时综合向量相似度、关键词、元数据和图谱关系进行排序。
5. 返回内容和溯源信息。

### 5.5 知识图谱

主要文件：

- `knowledge_graph.py`
- `neo4j_graph_store.py`

系统内置 438C 领域图谱，支持实体、关系和子图查询。Neo4j 可用时写入图数据库；不可用时保留本地 JSON 图谱能力。

### 5.6 文档合稿

文件：`document_merger.py`

职责：

- 多文档合并。
- 目录、标题层级、页眉页脚和样式处理。
- 导出 docx。
- 上传到 MinIO 或写入本地生成目录。

## 6. 数据模型概览

| 模型 | 说明 |
| --- | --- |
| Project | 项目基础信息 |
| KnowledgeBase | 知识库 |
| KnowledgeFile | 知识库文件、解析状态、审核状态 |
| Document | 438C 文档 |
| DocumentChapter | 文档章节内容 |
| GenerationTask | 文档生成任务 |
| MergeTask | 合稿任务 |
| ValidationRule | 验证规则 |
| ReviewRecord | 审核记录 |
| FileVersionHistory | 文件版本历史 |

## 7. 启动入口

后端入口：

```bash
cd backend
python run.py
```

前端入口：

```bash
cd frontend
npm run dev -- --host 0.0.0.0
```
