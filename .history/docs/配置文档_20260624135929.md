# 438C文档生成与管理系统 - 配置文档

## 1. 环境变量配置

系统通过 `.env` 文件管理环境变量，位于 `backend/.env`。

### 1.1 基础配置

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| FLASK_ENV | development | 运行环境: development / testing / production |
| SECRET_KEY | - | Flask密钥，生产环境必须修改为随机字符串 |

### 1.2 LLM配置

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| LLM_API_BASE | https://api.openai.com/v1 | LLM API地址，可替换为兼容OpenAI格式的本地/私有部署 |
| LLM_API_KEY | your-api-key-here | LLM API密钥 |
| LLM_MODEL | gpt-4 | 使用的模型名称 |

**本地部署LLM配置示例**（如使用vLLM/Ollama）:
```env
LLM_API_BASE=http://localhost:8000/v1
LLM_API_KEY=not-needed
LLM_MODEL=qwen2.5-72b-instruct
```

### 1.3 数据库配置

| 环境 | SQLAlchemy数据库URI |
|------|---------------------|
| 开发 | sqlite:///data/app.db |
| 测试 | sqlite:// |
| 生产 | postgresql://user:pass@localhost/docgen |

### 1.4 文件存储配置

| 配置项 | 默认路径 | 说明 |
|--------|----------|------|
| UPLOAD_FOLDER | data/uploads | 上传文件根目录 |
| GENERATED_FOLDER | data/generated | 生成文档目录 |
| VECTOR_DB_PATH | data/vector_db | ChromaDB向量数据库目录 |

## 3. 438C规则配置

### 3.1 标准文档类型 (`standard_documents.py`)

定义438C规定的20种文档类型，每种包含：

```python
"SRS": {
    "code": "SRS",
    "name": "软件需求规格说明",
    "category": "development",       # 分类: development/test/management
    "complexity": "high",            # 复杂度: high/medium/low
    "description": "...",
    "chapters": [                    # 章节结构
        {
            "number": "1",
            "title": "范围",
            "level": 1,
            "required": True,
            "sub_chapters": [...]
        },
        ...
    ]
}
```

## 4. 向量模型配置

### 4.1 Embedding模型

默认使用 `sentence-transformers` 的 `all-MiniLM-L6-v2` 模型。

如需更换模型，修改 `semantic_indexer.py` 中的初始化参数：

```python
# 使用本地模型
embedding_function = embedding_functions.SentenceTransformerEmbeddingFunction(
    model_name="shibing624/text2vec-base-chinese"  # 中文优化模型
)

# 使用OpenAI Embedding
embedding_function = embedding_functions.OpenAIEmbeddingFunction(
    api_key="your-key",
    model_name="text-embedding-3-small"
)
```

### 4.2 ChromaDB配置

```python
# 持久化存储
client = chromadb.PersistentClient(path="./data/vector_db")

# 内存模式（测试用）
client = chromadb.Client()
```

## 5. 文本预处理配置

### 5.1 敏感词库

系统内置军标敏感词库，支持自定义扩展：

```python
# 在 text_preprocessor.py 中配置
DEFAULT_SENSITIVE_WORDS = ['机密', '绝密', '秘密', '内部']

# 自定义词库文件（每行一个词）
sensitive_words_path = "config/sensitive_words.txt"
```

### 5.2 噪声模式

```python
NOISE_PATTERNS = [
    r'[\x00-\x08\x0b\x0c\x0e-\x1f\x7f]',  # 控制字符
    r'\u3000{2,}',                            # 连续全角空格
    r'[ \t]{4,}',                             # 连续半角空格
    r'\n{3,}',                                # 连续换行
    r'第\s*\d+\s*页\s*/\s*\d+\s*页',          # 页码标记
]
```

## 7. Rerank配置

在 `rerank_service.py` 中配置打分权重：

```python
# 综合得分权重
final_score = (
    semantic_score * 0.30 +     # 语义相似度
    military_score * 0.25 +     # 军标关键词匹配
    kg_score * 0.20 +           # 知识图谱关联
    metadata_score * 0.15 +     # 元数据匹配
    quality_score * 0.10        # 内容质量
)
```
